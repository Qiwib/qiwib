Output files

QiwiB creates a lot of output files. Some of them are located in the main output directory and others in its subdirectory time_dep/. They all can be read by any text editors (see the notes for gzipped files).

Main output files

.continue empty file which purpose it is to keep the simulation alive. Once deleted the simulation stops properly and safely after it finished its last time step
input an exact copy of the input file with a few additional lines appended to the end that contain the command line parameters with which QiwiB has been started
log logs certain properties in columns:
  1. time
  2. time step
  3. energy
  4. energy difference to previous time-step
  5. energy difference to previous time-step extrapolated to one time unit
  6. correlation between C(t) and C(t-dt)
  7. correlation between phi(t) and phi(t-dt) (phi being the matrix of the single-particle basis)
  8. normalisation of C
  9. normalisation of the total wave function
  10. [only for relaxation: error output (flags) of eigs function, see 'help eigs' in octave terminal, if the size of Hilbert space <=10 it is always 0]
The numbers in the brackets in the first few lines of the file indicate the column numbers.
phiC_restart.gz the most resent phi and C, used automatically for pa.relaxation=-2
To save disk space this file is gzipped, i.e. use gunzip if you want to see its contents in a text editor or use an editor that supports gzipped text files (some linux editors do).
pop_nat population of the natural orbitals as a percentage, first column is time
spatial_population integral of the density over the 3 regions defined by pa.nl and pa.nr
  1. time
  2. integral of density over all regions
  3. integral of density over left region
  4. integral of density over middle region
  5. integral of density over right region
steps saves the real cpu_time in seconds for each constant mean field (CMF) time step
  1. counts the number of elapsed mean field steps
  2. time step
  3. time
  4. real CPU times for
    • for last step
    • total time
    • calculation of W_ksql, h_kq and H_nonlinear (internal function) for last step
    • calculation of W_ksql, h_kq and H_nonlinear (internal function) - total time
    • calculation of rho_ksql, rho_kq and H_C for last step
    • calculation of rho_ksql, rho_kq and H_C - total time
    • calculation of phi propagation/relaxation for last step
    • calculation of phi propagation/relaxation - total time
    • calculation of C propagation/relaxation for last step
    • calculation of C propagation/relaxation - total time
The numbers in the brackets in the first few lines of the file indicate the column numbers.

Files in time_dep/ directory

The files in this directory are saved after each pa.save_step. [time] will be replaced by the current time in the simulation when the variables are due to be saved. Which files are saved is specified in pa.save_options. Also, the information contained in those files can be read out by a text editor or the plotting programs.
To save disk space all files are gzipped, i.e. use gunzip if you want to see their contents in a text editor or use an editor that supports gzipped text files (some linux editors do).

[time]_density.gz saves the density including the spatial grid, time and external potential
[time]_g1_0x.gz saves a slice of the reduced density matrix g1(0,x) including the spatial grid and the time
[time]_phiC.gz saves single-particle basis and the C vector including the spatial grid and the time
[time]_phiNO.gz saves natural orbitals r including the spatial grid and the time
[time]_rho.gz data necessary for plotting fock space of natural orbitals, g1 and g2
